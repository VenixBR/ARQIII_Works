--+ Makefile                        -- Makefile with commands to call Xcelium



In Makefile you can include the flags "GUI" and "TB". 
"GUI=1" will open the GUI of Xcelium.
"TB=0" do not will load to Xcelium the testbench archive.

Examples:
make CLA TB=0                   -- Will only test the sintax of code
make Top GUI=1                  -- Will load the testbench to test the circuit and open the GUI
make ControlPath GUI=1 TB=0     -- Will only test the sintax of code and open GUI
